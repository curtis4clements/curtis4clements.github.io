Open-Learning Collective: A List of Educational Resources

Education has gotten expensive, VERY expensive! I've been attempting to create a reference document to allow for exploration of FREE educational resources. Organized below by category are websites that can be explored below.


Below are different educational resources organized by their subject matter.

Computer Science

  • MIT OPEN COURSEWARE - https://www.ocw.mit.edu

    • About - Open education resource spanning multiple subjects, but specializes in technology

  • W3SCHOOLS - https://www.w3schools.com

    • About - Comprehensive and frequently updated resource for web design. Sets a golden standard as an asset.


Information Technology

  • Microsoft Learn - https://learn.microsoft.com/

    • About - Microsoft documentation along with certifications
